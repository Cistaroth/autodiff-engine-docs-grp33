class autodiff.operation.Operation

Bases: ABC

Abstract class for operations in the computation graph.

Subclasses should implement the _forward and _backward methods.

The _forward method should compute the output of the operation given the input variables.

The _backward method should compute the gradients of the input variables with respect to the output variable.

The _delete_gradient method should delete the gradients of the input variables.

_inputs

List of input variables.

Type:

list

__call__()

Calls the operation with the given input variables.

_delete_gradient()

Deletes the gradients of the input variables.

_forward()

Computes the output of the operation given the input variables.

_backward()

Computes the gradients of the input variables with respect to the output variable.

abstractmethod backward(incoming_gradient: ndarray) → None

Abstract method to implement the backward pass of the operation.

This method should compute the gradients of the input variables given the incoming gradient of the output variable. The gradients should be stored in the corresponding gradient attribute of the input variables.

Parameters:

incoming_gradient (np.ndarray) – The gradient of the output variable.

Returns:

None

delete_gradient() → None

Deletes the gradients of the input variables.

This method iterates over the input variables and calls their delete_gradient method to delete their gradients.

Returns:

None
