property parameters: dict[str, Variable]

Returns a dictionary containing the parameters of the model.

The dictionary contains a single key ‘parameters’ mapped to

a Variable object containing the parameters of the model.

The Variable object is a copy of the parameters stored in the model.

If the model has not been fitted, a warning is issued

and an empty dictionary is returned.

compute(inputs: list[Variable]) → Variable

Computes the output of the node.
